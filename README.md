# Knowledge-Testing-Quiz-Game
A simple and interactive Quiz Game developed in C using basic programming concepts like structures, arrays, and loops. This mini project allows users to play a multiple-choice quiz, view their score, and test their general knowledge in a fun way.
Technologies Used
Programming Language: C  
IDE/Compiler: GDB Online Compiler  
**Concepts Used:**  
Structures  
Arrays  
Loops and Conditional Statements  
Random Function (rand() from <stdlib.h>)  
Input/Output handling
